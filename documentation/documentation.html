Overview

Thank you for purchasing ARCHIVE_ROOT.

ARCHIVE_ROOT is a cinematic, terminal-inspired portfolio template built for developers, designers, and creatives who want a unique digital presence.

The template is fully responsive and built using modern front-end technologies including HTML5, CSS3, Bootstrap 5, JavaScript, GSAP, and PHPMailer.

Features

  • Fully Responsive Design
  • Terminal / Cyberpunk Aesthetic
  • Bootstrap 5 Grid System
  • GSAP Animations
  • ScrollTrigger Effects
  • Text Scramble Animations
  • Project Showcase Section
  • Animated Timeline
  • Sticky Footer Reveal
  • AJAX Contact Form
  • PHPMailer Integration
  • Clean and Organized Code
  • Easy Customization

Installation

Step 1

Extract the downloaded ZIP file.

Step 2

Upload the template files to your web server.

Step 3

Open index.html and begin customizing your content.

Step 4

Configure the contact form if you plan to receive emails through the website.

Contact Form Setup

Install PHPMailer

composer require phpmailer/phpmailer

Configure Email Settings

'to_email' => 'your-email@example.com',
'to_name' => 'Your Name',

'smtp_host' => 'smtp.gmail.com',
'smtp_port' => 587,
'smtp_username' => 'your-email@gmail.com',
'smtp_password' => 'your-app-password',

CORS Configuration

$allowed_origin = 'https://yourdomain.com';
If you use Gmail, you must generate an App Password and use it instead of your normal Google account password.
